# Skipton 264 Squadron Code 🚀

Welcome to the official code repository for **264 (Skipton) Squadron, Air Training Corps**. 

This repository serves as a central hub for projects developed by our cadets, specifically focusing on micro:bit development, Python scripting for Wing-level events, STEM competitions, and squadron training nights.

---

## 📖 About This Repo

The goal of this repository is to showcase the technical skills of our cadets and provide a library of reusable code for future squadron projects. Whether it's a micro:bit sensor for a field craft exercise or a data logger for a STEM challenge, you'll find our squadron's innovation here.

### Common Projects:
* **Micro:bit Python Scripts:** Interactive tools for navigation, communication, and environmental sensing.
* **Wing Event Code:** Specialist software written for inter-squadron competitions.
* **STEM Training:** Resources used during squadron training nights to teach coding fundamentals.

---

## 🛡️ Repository Rules & Safety

To maintain the safety and privacy of all our members, the following rules are strictly enforced:

1. **No Personal Data:** No personally identifiable names, addresses, phone numbers, or social media handles are to be included in the code, comments, or commit messages.
2. **Standard Naming:** Use generic terms (e.g., `Cadet1`, `TeamAlpha`) instead of real names.
3. **Appropriate Content:** All code and text must align with the values of the Royal Air Force Air Cadets (RAFAC). No offensive language or "easter eggs" allowed.
4. **Supervision:** All hardware-based projects (involving batteries or external circuits) should be tested under the supervision of a Squadron Staff member or qualified CI.

---

## 💻 Contribution Guidelines

Cadets are encouraged to contribute! When submitting code:
* Ensure your code is well-commented so other cadets can understand how it works.
* Include a brief description of what the code does at the top of the file.
* Test your code on the physical hardware (e.g., the micro:bit) before pushing.

---

## 🛡️ Safeguarding & Digital Citizenship

Skipton 264 Squadron is committed to the safety of our young people. In addition to the "No PII" rule, we expect all contributors to practice good digital citizenship:
* Be supportive in code reviews.
* Credit others if you have adapted their code.
* Use this platform to learn and help others improve.
